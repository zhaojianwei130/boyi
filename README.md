该目录存储一个eduStack edX实例的默认主题。


主题创作
===============
定制你的主题:
- 叉这个存储库。
- 克隆到的主题目录在你edx-platform目录,重命名主题目录到您的新主题的名称。
- 上传自己的图片资产。
- 编辑。scss文件静态/ sass /和重命名文件和你的主题的名称。
- 编辑lms.envs。在edx-platform json文件,设置“USE_CUSTOM_THEME”真,和“THEME_NAME”主题的名字。
